# Project Brief

## Overview
Kilo Code is a VS Code AI coding assistant with persistent project memory. Provides $20 free credits for Claude 4 and Gemini 2.5 Pro models.

## Core Features
- Code generation and refactoring
- Terminal and browser automation
- Memory Bank for persistent context
- Multi-mode system (Architect, Code, Debug)
- MCP servers for extended capabilities
- Tree-sitter code parsing for 30+ languages

## Development
- Monorepo with pnpm (NEVER use npm)
- TypeScript, Node.js v20.18.1
- Run tests: `pnpm test`, `pnpm test --filter=<package>`, `pnpm test -- <file>`
