# Product Overview

## Problem
Traditional coding assistants lack persistent project context and provide generic solutions that don't align with project-specific patterns.

## Solution
Kilo Code provides AI coding assistance with persistent memory through Memory Bank, specialized modes, and integration with external tools.

## Goals
- Maintain project context across sessions
- Provide project-specific code suggestions
- Support multiple languages and frameworks

## Users
- Developers of all experience levels working with complex projects
