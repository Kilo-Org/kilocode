# Bienvenue dans le projet Kilocode

Voir [la Documentation pour les détails sur les contributions](https://kilocode.ai/docs/extending/contributing-to-kilo)

## Résumé

Il existe de nombreuses façons de contribuer au projet

- **Contributions de code** Implémenter de nouvelles fonctionnalités ou corriger des bugs
- **Documentation :** Améliorer la documentation existante ou créer de nouveaux guides
- **Modes personnalisés :** Créer et partager des modes spécialisés
- **Rapports de bugs :** Signaler les problèmes que vous rencontrez
- **Demandes de fonctionnalités :** Suggérer de nouvelles fonctionnalités ou améliorations
- **Support communautaire :** Aider les autres utilisateurs dans la communauté

La communauté Kilocode est [ici sur Discord](https://kilocode.ai/discord)
