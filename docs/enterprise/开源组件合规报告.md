## 6. 扫描命令（复现）

```powershell
cd D:\ai\kilocode-main
bun install

# 各 workspace 摘要
foreach ($p in @('packages/kilo-vscode','packages/opencode','packages/kilo-ui','packages/kilo-gateway')) {
  Write-Host "`n=== $p ==="
  pushd $p
  bunx license-checker --production --summary
  popd
}

# GPL 族排查（应为 0）
pushd packages\kilo-vscode
bunx license-checker --production --json | bun -e "const d=JSON.parse(require('fs').readFileSync(0,'utf8')); console.log(Object.entries(d).filter(([,v])=>/GPL|LGPL|AGPL|SSPL/i.test(v.licenses||'')).length)"
popd

# Go 桥接（无第三方 module）
type deploy\enterprise\bridge\go.mod
```

**维护：** 每次 **VSIX / Engine 镜像发版** 或 **npm/go 重大升级** 后复跑 §6 并更新 §3、§5 日期。
