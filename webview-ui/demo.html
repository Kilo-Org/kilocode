📱 配置界面预览

新增的MessageSendingSettings组件提供了直观的配置界面，包含以下主要区域：

预设模板选择

提供三种预设模板，快速配置不同的优化策略

  • 最小化模板 - 最大化Token节省
  • 标准模板 - 平衡性能与功能
  • 完整模板 - 保持完整上下文
  • 自定义模板 - 手动精细配置

内容包含选项

精确控制消息中包含的内容类型

  • 系统提示词控制
  • 对话历史管理
  • 环境详情包含
  • 工具描述优化
  • MCP状态信息

历史消息配置

智能管理对话历史，优化上下文长度

  • 历史消息数量限制
  • 自动压缩功能
  • 压缩阈值设置
  • 智能摘要生成

性能优化选项

多维度性能优化，提升整体效率

  • 上下文缓存机制
  • 图片自动优化
  • Token智能压缩
  • 内容去重处理

配置示例

以下是标准模板的配置示例：

{
  "enabled": true,
  "template": "standard",
  "includeSystemPrompt": true,
  "includeConversationHistory": true,
  "includeEnvironmentDetails": true,
  "includeToolDescriptions": true,
  "includeMcpStatus": true,
  "includeRooIgnoreInstructions": true,
  "maxHistoryMessages": 20,
  "historyCompressionEnabled": false,
  "historyCompressionThreshold": 10,
  "systemPromptTemplate": "full",
  "enableContextCaching": false,
  "enableImageOptimization": false,
  "enableTokenOptimization": false
}
