- Search the web using Perplexity Search API for current documentation, changelogs, releases, errors, and other recent information
- Returns concise grounded results with title, URL, snippet, and date
- Use this tool for information beyond your knowledge cutoff or when you need source URLs
- Results are retrieved in a single API call and are optimized for coding-agent research

Usage notes:
  - `max_results` controls how many results to return. Use a small number for targeted lookups and a larger number when comparing sources.
  - `country` can localize results with an ISO 3166-1 alpha-2 country code.
  - If Perplexity credentials are not configured, the tool will return a clear setup message instead of crashing.

The current year is {{year}}. You MUST use this year when searching for recent information or current events
- Example: If the current year is 2026 and the user asks for "latest AI news", search for "AI news 2026", NOT "AI news 2025"
