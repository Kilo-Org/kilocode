# Security Agent Audit Reports

Security Agent Audit Reports provide period-bounded evidence of material Security Finding actions and outcomes for investigation and compliance work. Audit reports capture Security Finding Activity Events including sync, analysis, dismissal, remediation, and deletion actions.
