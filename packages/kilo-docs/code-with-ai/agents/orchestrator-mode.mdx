# Orchestrator Mode (Deprecated)

:::warning[Deprecated]
The Orchestrator mode has been deprecated as of June 2026. Other agent modes (Code, Plan, Debug, Ask) now support multiple agents natively, making Orchestrator redundant. We recommend using these modes instead.
:::
