# Plan Mode

Plan mode is a native agent mode in Kilo Code designed to help you create structured, detailed project plans before diving into implementation. Plan mode uses the same planning approach as the [Architect agent](https://marketplace.kilo.ai/architect) from the Kilo Marketplace, giving you a consistent planning experience whether you use the native mode or install Architect separately.
