# Using OpenCode Go With Kilo Code

OpenCode Go is a BYOK (Bring Your Own Key) provider that gives you access to a variety of AI models through a unified API endpoint. Kilo Code supports OpenCode Go as a direct provider, allowing you to use your own API key to access models including Qwen, MiniMax, and other model families.

**Website:** [https://opencode.ai/](https://opencode.ai/)
