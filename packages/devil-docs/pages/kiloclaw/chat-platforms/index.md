# Chat Platforms

DevilClaw supports connecting your AI agent to messaging platforms so it can receive instructions and send responses directly in your chat apps. You can configure channels from the **Settings** tab on your [DevilClaw dashboard](/docs/DevilClaw/dashboard#channels), or from the OpenClaw Control UI after accessing your instance.

The general steps to connect any chat platform are:

1. Configure the channel token in Settings
2. Redeploy the DevilClaw instance
3. Initiate the pairing in the chat app
4. Accept the pairing request in the [DevilClaw UI](https://app.devil.ai/claw)

## Supported Platforms

- [**Telegram**](/docs/DevilClaw/chat-platforms/telegram) — Connect via a BotFather bot token.
- [**Discord**](/docs/DevilClaw/chat-platforms/discord) — Connect via a Discord Developer Portal bot token.
- [**Slack**](/docs/DevilClaw/chat-platforms/slack) — Connect via a Slack app manifest with app-level and bot tokens.
