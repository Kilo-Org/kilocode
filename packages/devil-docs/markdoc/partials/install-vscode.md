### Install directly

1. If you don't have VS Code installed, download it from [code.visualstudio.com](https://code.visualstudio.com/)
2. Then, you can click the button below to install devil Code directly from the VS Code Marketplace:

[![Install devil Code](https://raster.shields.io/badge/Install%20Devil%20Code-F8F674?style=for-the-badge)](vscode:extension/devilcode.devil-code)

### Install from VS Code Marketplace

1. Open VS Code
2. Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS) to open Extensions
3. Search for "devil Code"
4. Click **Install**

### Install via Command Line

```bash
code --install-extension devilcode.devil-code
```

### Verify Installation

After installation, you should see the devil Code icon ({% devilcodeIcon /%}) in the Activity Bar on the left side of VS Code. Click it to open the devil Code panel.
