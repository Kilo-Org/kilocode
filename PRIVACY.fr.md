# Politique de confidentialité de Kilo Code

**Dernière mise à jour : 7 mars 2025**

Kilo Code respecte votre vie privée et s'engage à être transparent quant à la manière dont nous traitons vos données. Ci-dessous figure une explication simple de l'endroit où les principales données sont envoyées - et surtout, où elles ne le sont pas.

### **Où vos données vont (Et où elles ne vont pas)**

- **Code & Fichiers** : Kilo Code accède aux fichiers sur votre machine locale lorsque cela est nécessaire pour les fonctionnalités assistées par l'IA. Lorsque vous envoyez des commandes à Kilo Code, des fichiers pertinents peuvent être transmis au fournisseur de modèle d'IA de votre choix (par exemple, OpenAI, Anthropic, OpenRouter) pour générer des réponses. Nous n'avons pas accès à ces données, mais les fournisseurs d'IA peuvent les stocker conformément à leurs politiques de confidentialité.

- **Commandes** : Toutes les commandes exécutées via Kilo Code ont lieu dans votre environnement local. Cependant, lorsque vous utilisez des fonctionnalités alimentées par l'IA, le code et le contexte pertinents de vos commandes peuvent être transmis au fournisseur de modèle d'IA de votre choix (par exemple, OpenAI, Anthropic, OpenRouter) pour générer des réponses. Nous n'avons pas accès à ces données ni ne les stockons, mais les fournisseurs d'IA peuvent les traiter conformément à leurs politiques de confidentialité.

- **Invitations & Demandes à l'IA** : Lorsque vous utilisez des fonctionnalités alimentées par l'IA, vos invitations et le contexte de projet pertinent sont envoyés au fournisseur de modèle d'IA de votre choix (par exemple, OpenAI, Anthropic, OpenRouter) pour générer des réponses. Nous ne stockons ni ne traitons ces données. Ces fournisseurs d'IA ont leurs propres politiques de confidentialité et peuvent stocker des données conformément à leurs conditions d'utilisation.

- **Clés API & Identifiants** : Si vous saisissez une clé API (par exemple, pour connecter un modèle d'IA), elle est stockée localement sur votre appareil et n'est jamais envoyée à notre service ni à un tiers, sauf au fournisseur que vous avez choisi.

### **Vos choix & Contrôle**

- Vous pouvez exécuter des modèles localement pour éviter l'envoi de données à des tiers.

### **Sécurité & Mises à jour**

Nous prenons des mesures raisonnables pour sécuriser vos données, mais aucun système n'est sécurisé à 100 %. Si notre politique de confidentialité change, nous vous en informerons dans l'extension.

### **Contactez-nous**

Pour toute question relative à la confidentialité, vous pourrez bientôt nous joindre à l'adresse support@kilocode.ai.

---

En utilisant Kilo Code, vous acceptez la présente Politique de confidentialité.
