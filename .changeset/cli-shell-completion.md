Add shell completion command for bash, zsh, fish, and powershell

Usage:
- `kilocode completion bash >> ~/.bashrc`
- `kilocode completion zsh >> ~/.zshrc`
- `kilocode completion fish > ~/.config/fish/completions/kilocode.fish`
- `kilocode completion powershell >> $PROFILE`

Supports both `kilocode` and `kilo` command aliases.
