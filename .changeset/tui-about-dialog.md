Add an About dialog in the TUI accessible from the command palette (under "Kilo") or via the `/about` slash command. Shows version, channel, build kind, license, runtime, platform, $TERM, config path, project root, cwd, connected providers, default model, and agent/MCP/plugin counts, with helpful links and a `c` shortcut to copy diagnostics to the clipboard.
