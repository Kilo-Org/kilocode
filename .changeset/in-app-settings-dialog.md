Add an in-TUI settings dialog (palette command "settings" / Ctrl+O) that manages provider connections, models, default agent, plugins, and terminal preferences without editing config files by hand. Includes a searchable settings list with project/global scope, immediate-save toggles, sub-dialogs for theme/diff layout/title icon/scroll speed/model pickers, provider disconnect and disable, a Disabled providers list for re-enabling providers hidden by `disabled_providers`, plugin enable/disable, auto-approve toggle, and minor TUI polish (visible scrollbar opt-in, busy/error handling on the provider connect prompt, dialog option truncation).
