## Task 2: Aggregation Endpoint

Create the workflow aggregation endpoint that computes telemetry metrics from the event log.

**Action:**
1. Create `packages/opencode/src/devilcode/workflow/aggregations.ts` with:
   ```typescript
   export interface AggregationResponse {
     successRateByTeam: Record<string, { completed: number; started: number; rate: number }>
     stallRateByPosition: Record<string, { maxWaitMs: number; avgWaitMs: number }>
     costByWorkflow: Array<{ workflowId: string; totalCost: number }>
     durationByStage: Record<string, { avgMs: number; p95Ms: number; count: number }>
     generatedAt: string
   }
   
   export async function computeAggregations(planningDir: string): Promise<AggregationResponse>
   ```
2. Implement `computeAggregations()`:
   - Use EventLogger.readAll() to get all events
   - Compute success rate: count task_completed vs task_started per team (from metadata.teamId or "default")
   - Compute stall rate: track time between task_started and task_completed per role
   - Compute cost: sum metadata.cost if present (default to 0)
   - Compute duration: group by stage, calculate avg and p95
3. Add route in `packages/opencode/src/devilcode/workflow/routes.ts`:
   - `GET /devilcode/workflow/aggregations` — returns AggregationResponse
   - **CRITICAL (R4):** Use `Instance.directory` to resolve planning directory:
     ```typescript
     const planningDir = path.join(Instance.directory, ".planning")
     const aggregations = await computeAggregations(planningDir)
     ```
   - This matches existing pattern in routes.ts (see WorkflowStateManager usage)
4. Handle empty event log gracefully (return zero-initialized response)
5. **WARNING (R5):** Add optional query params for performance:
   - `?since=ISO_DATE` — filter events after date (optional, no default)
   - `?limit=N` — limit events processed (default: 10000)
   - **CYCLE 2 (R5 compat):** Default limit=10000 applies automatically when param omitted
   - Existing callers without query params get safe behavior (capped at 10000 events)
   - Document: "For event logs >10MB, consider archiving old events"

> verification: `test -f packages/opencode/src/devilcode/workflow/aggregations.ts` returns 0
> verification: `grep -n "computeAggregations" packages/opencode/src/devilcode/workflow/aggregations.ts` returns line number
> verification: `grep -n "/aggregations" packages/opencode/src/devilcode/workflow/routes.ts` returns line number

**Done:** Aggregation endpoint returns computed metrics JSON.
