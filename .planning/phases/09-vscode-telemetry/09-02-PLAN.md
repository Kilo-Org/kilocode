## Task 3: TeamBuilderHandler Class

Create isolated handler for team builder messages to avoid bloating AgentManagerProvider.

**Action:**
1. Create `packages/devil-vscode/src/agent-manager/team-builder-handler.ts`:
   ```typescript
   export class TeamBuilderHandler {
     constructor(
       private connectionService: DevilConnectionService,
       private postMessage: (msg: unknown) => void,
       private log: (msg: string) => void // Output channel logging
     ) {}
     
     async handleMessage(msg: AgentManagerInMessage): Promise<boolean> {
       // Returns true if handled, false if not a team builder message
     }
     
     private async handleLoadTeam(teamId: string): Promise<void>
     private async handleListTeams(): Promise<void>
     private async handleSaveTeam(teamId: string, config: unknown): Promise<void>
     private async handleDeleteTeam(teamId: string): Promise<void>
     private async handleGetAggregations(): Promise<void>
   }
   ```
2. **CRITICAL (R3):** Implement each handler with proper error handling:
   ```typescript
   private async handleLoadTeam(teamId: string): Promise<void> {
     try {
       const team = await this.connectionService.getTeam(teamId)
       this.postMessage({ type: "teamBuilder.teamLoaded", teamId, config: team })
     } catch (err) {
       const message = err instanceof Error ? err.message : "Unknown error"
       this.log(`[TeamBuilder] Failed to load team ${teamId}: ${message}`)
       this.postMessage({ 
         type: "teamBuilder.error", 
         code: "LOAD_FAILED", 
         teamId,
         message 
       })
     }
   }
   ```
   - Every handler MUST wrap in try/catch
   - Log to output channel on error
   - Send `teamBuilder.error` message to webview
   - Include error code (LOAD_FAILED, SAVE_FAILED, DELETE_FAILED, AGGREGATION_FAILED)
3. Wire into `AgentManagerProvider.handleMessage()`:
   - Import TeamBuilderHandler
   - Initialize in constructor or lazily
   - Check `teamBuilderHandler.handleMessage(msg)` first in handleMessage()
   - If returns true, return null (handled)
