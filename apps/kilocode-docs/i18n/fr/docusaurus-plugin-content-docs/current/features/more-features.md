# Fonctionnalités Additionnelles

Les extras de Kilo Code rationalisent les tâches de routine et améliorent l'accessibilité.

## Réponses Suggérées

Kilo Code offre des réponses suggérées pour que vous passiez moins de temps à taper.

- Après que vous posiez une question, des boutons apparaissent sous la boîte de chat.
- Cliquez sur un bouton pour le réutiliser comme votre prochain prompt.

## Texte à Parole

La fonctionnalité Texte à Parole permet à Kilo Code de lire les réponses à haute voix.

1. Activez TTS dans les paramètres.
2. Cliquez sur l'icône haut-parleur à côté de n'importe quelle réponse pour commencer à écouter.

## Support de Langue Global

Kilo Code supporte 14 langues :

- Chinois Simplifié
- Chinois Traditionnel
- Espagnol
- Hindi
- Français
- Portugais
- Allemand
- Japonais
- Coréen
- Italien
- Turc
- Vietnamien
- Polonais
- Catalan

Changez les langues sous **Paramètres Avancés > Langue**.
