# Vue d'ensemble des Fournisseurs d'API

**En résumé** : Kilo Code, comme tout outil de codage IA agent, a besoin de fournisseurs de modèles IA pour fonctionner. Vous pouvez utiliser notre [fournisseur intégré](/providers/kilocode) (le plus simple) ou connecter vos propres clés API de **30+ fournisseurs**.

## Extension Kilo Code vs Fournisseur d'API

### L'Extension

- Outil VS Code que vous installez depuis la marketplace
- Gère l'interface utilisateur, les opérations de fichiers et la gestion des flux de travail
- [Open source](https://github.com/Kilo-Org/kilocode)
- Se connecte à n'importe quel fournisseur IA

### Fournisseur Kilo Gateway

- **Option intégrée** qui vient avec l'extension
- Connexion Google avec crédits gratuits inclus
- Aucune gestion de clé API requise
- Accès aux modèles de codage de pointe
- [Tarification compétitive](https://kilocode.ai/pricing)

**Point clé** : L'extension fonctionne avec n'importe quel fournisseur — notre service API est juste l'option "tout inclus".

## Premiers Pas : Deux Chemins

### Option 1 : Fournisseur Intégré (Recommandé)

✅ **Configuration la plus rapide**

- Connectez-vous avec Google ou GitHub sur kilocode.ai
- Crédits gratuits inclus pour votre première recharge
- Zéro gestion de clé API
- Derniers modèles disponibles

→ [Guide de configuration complet](/providers/kilocode)

### Option 2 : Votre Propre Fournisseur

**Plus de contrôle, plus de configuration**

1. Choisissez parmi 30+ fournisseurs pris en charge
2. Obtenez une clé API de votre fournisseur
3. Configurez dans les paramètres de Kilo Code

## Pourquoi Utiliser Plusieurs Fournisseurs ?

- **Coût** : Comparez les tarifs entre les fournisseurs
- **Modèles** : Accédez à différentes capacités IA
- **Fiabilité** : Options de secours pour les pannes
- **Fonctionnalités** : Certains fournisseurs offrent des modèles exclusifs
- **Régional** : Meilleures performances dans certaines zones
