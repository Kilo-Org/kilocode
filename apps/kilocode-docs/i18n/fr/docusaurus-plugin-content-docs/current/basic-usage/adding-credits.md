# Ajouter plus de Kilo Credits

Une fois que vous avez utilisé vos Kilo Credits initiaux gratuits, vous pouvez facilement en ajouter d'autres :

- Ajoutez une carte de crédit à votre compte (tous les paiements sont traités de manière sécurisée via Stripe)
- Nous n'offrons actuellement pas d'abonnement, vous ne payez donc que pour les crédits que vous utilisez, et uniquement lorsque vous choisissez de recharger vos crédits !
- Profitez de votre propre fournisseur d'API en [connectant votre propre fournisseur d'API](/getting-started/connecting-api-provider.md)

## Tarification Transparente

Chez Kilo Code, nous croyons en une transparence totale des prix :

- Notre tarification correspond exactement aux tarifs API du fournisseur de modèles
- Nous ne prenons aucune commission ni majoration.
- 1 $ que vous nous donnez devient 1 $ de Kilo credits
- Nous débitons vos Kilo credits exactement de ce que le fournisseur nous facture en dollars
- Vous ne payez que ce que vous utilisez sans frais cachés
