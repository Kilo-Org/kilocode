# Utiliser Unbound avec Kilo Code

Kilo Code supporte l'accès aux modèles via [Unbound](https://getunbound.ai/), une plateforme qui se concentre sur la fourniture d'un accès sécurisé et fiable à une variété de modèles de langage volumineux (LLMs). Unbound agit comme une passerelle, vous permettant d'utiliser des modèles de fournisseurs comme Anthropic et OpenAI sans avoir besoin de gérer plusieurs clés API et configurations directement. Ils mettent l'accent sur les fonctionnalités de sécurité et de conformité pour l'utilisation d'entreprise.

**Site Web :** [https://getunbound.ai/](https://getunbound.ai/)

## Créer un Compte

1.  **S'inscrire/Se connecter :** Allez à la [passerelle Unbound](https://gateway.getunbound.ai). Créez un compte ou connectez-vous.
2.  **Créer une Application :** Allez à la page [Applications](https://gateway.getunbound.ai/ai-gateway-applications) et appuyez sur le bouton "Créer Application".
3.  **Copier la Clé API :** Copiez la clé API dans votre presse-papiers.

## Modèles Supportés

Unbound vous permet de configurer une liste de modèles supportés dans votre application, et Kilo Code récupérera automatiquement la liste des modèles disponibles depuis l'API Unbound.

## Configuration dans Kilo Code

1.  **Ouvrir les Paramètres Kilo Code :** Cliquez sur l'icône d'engrenage (<Codicon name="gear" />) dans le panneau Kilo Code.
2.  **Sélectionner le Fournisseur :** Choisissez "Unbound" dans le menu déroulant "Fournisseur API".
3.  **Saisir la Clé API :** Collez votre clé API Unbound dans le champ "Clé API Unbound".
4.  **Sélectionner le Modèle :** Choisissez votre modèle désiré dans le menu déroulant "Modèle".

## Conseils et Notes

- **Focus Sécurité :** Unbound met l'accent sur les fonctionnalités de sécurité pour l'utilisation d'entreprise. Si votre organisation a des exigences de sécurité strictes pour l'utilisation d'IA, Unbound pourrait être une bonne option.
