# Utiliser Fireworks AI avec Kilo Code

Fireworks AI est une plateforme haute performance pour l'exécution de modèles d'IA qui offre un accès rapide à une large gamme de modèles de langage open-source et propriétaires. Construit pour la vitesse et la fiabilité, Fireworks AI fournit à la fois des options de déploiement serverless et dédiées avec des APIs compatibles OpenAI.

**Site Web :** [https://fireworks.ai/](https://fireworks.ai/)

---

## Obtenir une Clé API

1. **S'inscrire/Se connecter :** Allez à [Fireworks AI](https://fireworks.ai/) et créez un compte ou connectez-vous.
2. **Naviguer vers les Clés API :** Après vous être connecté, allez à la [page Clés API](https://app.fireworks.ai/settings/users/api-keys) dans les paramètres de compte.
3. **Créer une Clé :** Cliquez sur "Créer une clé API" et donnez à votre clé un nom descriptif (par ex., "Kilo Code").
4. **Copier la Clé :** Copiez la clé API _immédiatement_ et stockez-la de manière sécurisée. Vous ne pourrez plus la voir.

---

## Modèles Supportés

Kilo Code supporte les modèles Fireworks AI suivants :

- `accounts/fireworks/models/kimi-k2-instruct` - Modèle d'instructions Kimi K2
- `accounts/fireworks/models/qwen3-235b-a22b-instruct-2507` - Modèle d'instructions Qwen 3 235B
- `accounts/fireworks/models/qwen3-coder-480b-a35b-instruct` - Qwen 3 Coder 480B pour la génération de code
- `accounts/fireworks/models/deepseek-r1-0528` - Modèle de raisonnement DeepSeek R1
- `accounts/fireworks/models/deepseek-v3` - Modèle de génération DeepSeek V3

---

## Configuration dans Kilo Code

1. **Ouvrir les Paramètres Kilo Code :** Cliquez sur l'icône d'engrenage (<Codicon name="gear" />) dans le panneau Kilo Code.
2. **Sélectionner le Fournisseur :** Choisissez "Fireworks AI" dans le menu déroulant "Fournisseur API".
3. **Saisir la Clé API :** Collez votre clé API Fireworks AI dans le champ "Clé API Fireworks AI".
4. **Sélectionner le Modèle :** Choisissez votre modèle désiré dans le menu déroulant "Modèle".

---

## Conseils et Notes

- **Performance :** Fireworks AI est optimisé pour la vitesse et offre d'excellentes performances pour les tâches de chat et de complétion.
- **Tarification :** Référez-vous à la page [Tarification Fireworks AI](https://fireworks.ai/pricing) pour les informations de tarification actuelles.
- **Limites de Débit :** Fireworks AI a des limites de débit basées sur l'utilisation. Surveillez votre utilisation dans le tableau de bord et considérez mettre à jour votre plan si nécessaire.
