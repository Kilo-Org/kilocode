# Utiliser v0 avec Kilo Code

Kilo Code supporte v0, le fournisseur de modèle d'IA de Vercel qui offre une API compatible OpenAI. Ceci vous permet d'utiliser les modèles de v0 avec Kilo Code à travers l'interface d'API OpenAI familière.

## Prérequis

Pour utiliser v0 avec Kilo Code, vous aurez besoin :

- Un compte d'équipe avec Vercel v0
- Une clé API v0

## Configuration

Configurer v0 dans Kilo Code est simple :

1. Dans les paramètres Kilo Code (cliquez sur l'icône <Codicon name="gear" />) :
    - Sous **Fournisseur API**, sélectionnez : **OpenAI Compatible**
    - Définissez l'**URL de Base**: `https://api.v0.dev/v1`
    - Collez votre clé API v0
    - Définissez l'**ID de Modèle**: `v0-1.0-md`
    - Cliquez **Vérifier** pour confirmer la connexion

## Modèles Supportés

Actuellement, v0 offre le modèle suivant :

- `v0-1.0-md` - Le modèle multimodal de v0 qui supporte à la fois les entrées texte et image
- `v0-1.5-lg` - modèle composite ([en savoir plus](https://vercel.com/blog/v0-composite-model-family))
- `v0-1.5.md` - modèle composite ([en savoir plus](https://vercel.com/blog/v0-composite-model-family))

## Dépannage

- **"Clé API Invalide" :** Vérifiez deux fois que vous avez saisi la clé API correctement.
- **"Modèle Non Trouvé" :** Assurez-vous que vous utilisez l'ID de modèle correct (`v0-1.0-md`).
- **Erreurs de Connexion :** Vérifiez que l'URL de Base est correcte (`https://api.v0.dev/v1`).
- **Problèmes d'Accès :** Confirmez que votre compte d'équipe Vercel v0 est actif et correctement configuré.
