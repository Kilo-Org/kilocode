# Utiliser DeepSeek avec Kilo Code

Kilo Code supporte l'accès aux modèles via l'API DeepSeek, incluant `deepseek-chat` et `deepseek-reasoner`.

**Site Web :** [https://platform.deepseek.com/](https://platform.deepseek.com/)

## Obtenir une Clé API

1.  **S'inscrire/Se connecter :** Allez à la [Plateforme DeepSeek](https://platform.deepseek.com/). Créez un compte ou connectez-vous.
2.  **Naviguer vers les Clés API :** Trouvez vos clés API dans la section [Clés API](https://platform.deepseek.com/api_keys) de la plateforme.
3.  **Créer une Clé :** Cliquez sur "Créer une nouvelle clé API". Donnez à votre clé un nom descriptif (par ex., "Kilo Code").
4.  **Copier la Clé :** **Important :** Copiez la clé API _immédiatement_. Vous ne pourrez plus la voir. Stockez-la de manière sécurisée.

## Modèles Supportés

Kilo Code supporte les modèles DeepSeek suivants :

- `deepseek-chat` (Recommandé pour les tâches de codage)
- `deepseek-reasoner` (Recommandé pour les tâches de raisonnement)

## Configuration dans Kilo Code

1.  **Ouvrir les Paramètres Kilo Code :** Cliquez sur l'icône d'engrenage (<Codicon name="gear" />) dans le panneau Kilo Code.
2.  **Sélectionner le Fournisseur :** Choisissez "DeepSeek" dans le menu déroulant "Fournisseur API".
3.  **Saisir la Clé API :** Collez votre clé API DeepSeek dans le champ "Clé API DeepSeek".
4.  **Sélectionner le Modèle :** Choisissez votre modèle désiré dans le menu déroulant "Modèle".

## Conseils et Notes

- **Tarification :** Référez-vous à la page [Tarification DeepSeek](https://api-docs.deepseek.com/quick_start/pricing/) pour les détails sur les coûts des modèles.
