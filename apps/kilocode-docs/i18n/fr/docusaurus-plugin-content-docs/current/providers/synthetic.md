# Utiliser Synthetic avec Kilo Code

Synthetic fournit l'accès à plusieurs modèles d'IA open-source fonctionnant sur une infrastructure sécurisée aux États-Unis et en UE. Ils offrent à la fois des options de tarification basées sur l'abonnement et basées sur l'utilisation, avec de fortes garanties de confidentialité - ils ne s'entraînent jamais sur vos données et suppriment automatiquement les données API dans les 14 jours.

**Site Web :** [https://synthetic.new](https://synthetic.new)

## Obtenir une Clé API

1. **S'inscrire/Se connecter :** Allez à [Synthetic](https://synthetic.new) et créez un compte ou connectez-vous.
2. **Naviguer vers les Clés API :** Après vous être connecté, allez à la [page Clés API](https://synthetic.new/user-settings/api) dans vos paramètres de compte.
3. **Copier votre Clé :** Cliquez sur l'icône Copier à côté de votre clé pour la copier dans votre presse-papiers.

## Modèles Supportés

Kilo Code supporte tous les modèles d'IA Synthetic "always on". Les modèles disponibles incluent diverses options open-source optimisées pour différents cas d'utilisation.

**Note :** La disponibilité des modèles peut changer. Référez-vous à la [documentation Synthetic](https://synthetic.new) pour la liste la plus à jour des modèles supportés et leurs capacités.

## Configuration dans Kilo Code

1. **Ouvrir les Paramètres Kilo Code :** Cliquez sur l'icône d'engrenage (<Codicon name="gear" />) dans le panneau Kilo Code.
2. **Sélectionner le Fournisseur :** Choisissez "Synthetic" dans le menu déroulant "Fournisseur API".
3. **Saisir la Clé API :** Collez votre clé API Synthetic dans le champ "Clé API Synthetic".
4. **Sélectionner le Modèle :** Choisissez votre modèle désiré dans le menu déroulant "Modèle".

## Conseils et Notes

- **Options de Tarification :** Synthetic offre à la fois des abonnements et une tarification d'utilisation à la demande [tarification](https://synthetic.new/pricing).
- **Confidentialité :** Politique de confidentialité forte avec aucun entraînement sur les données utilisateur et suppression automatique des données API dans les 14 jours.
- **Compatibilité OpenAI :** Les modèles Synthetic fonctionnent avec n'importe quel outil et application compatible OpenAI.
