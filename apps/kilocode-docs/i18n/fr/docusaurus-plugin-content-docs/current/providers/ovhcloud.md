# Utiliser OVHcloud AI Endpoints avec Kilo Code

OVHcloud est un fournisseur Cloud français leader en Europe avec souveraineté de données et confidentialité.

Accédez facilement aux modèles d'IA pré-entraînés reconnus mondialement. Innovez en utilisant des APIs simples et sécurisées sur l'infrastructure robuste et axée sur la confidentialité d'OVHcloud. Améliorez vos applications avec des capacités d'IA évolutives, éliminant le besoin d'expertise approfondie. Accomplissez davantage avec des endpoints d'IA puissants conçus pour la simplicité et la fiabilité.

**Site Web :** [https://endpoints.ai.cloud.ovh.net](https://endpoints.ai.cloud.ovh.net)

:::info

Vous pouvez signaler tout bogue ou retour d'information en discutant avec nous sur notre [serveur Discord](https://discord.gg/ovhcloud), dans le canal AI Endpoints.

:::

## Obtenir une Clé API

1.  **S'inscrire/Se connecter :** Allez au [gestionnaire OVHcloud](https://www.ovh.com/manager). Créez un compte ou connectez-vous.
2.  **Naviguer vers le Cloud Public :** Allez à la section Cloud Public, et créez un nouveau projet. Naviguez vers AI Endpoints dans la section _IA & Machine Learning_.
3.  **Créer une Clé :** Cliquez vers _Clés API_ et créez une nouvelle clé.

## Modèles Supportés

Kilo Code récupérera automatiquement nos modèles depuis notre API. Il sera à jour.

Vous pouvez parcourir notre [catalogue](https://endpoints.ai.cloud.ovh.net/catalog) pour voir un détail de tous nos modèles.

## Configuration dans Kilo Code

1.  **Ouvrir les Paramètres Kilo Code :** Cliquez sur l'icône d'engrenage (<Codicon name="gear" />) dans le panneau Kilo Code.
2.  **Sélectionner le Fournisseur :** Choisissez "OVHcloud AI Endpoints" dans le menu déroulant "Fournisseur API".
3.  **Saisir la Clé API :** Collez votre clé API AI Endpoints dans le champ "Clé API OVHcloud AI Endpoints".
4.  **Sélectionner le Modèle :** Choisissez votre modèle désiré dans le menu déroulant "Modèle".
