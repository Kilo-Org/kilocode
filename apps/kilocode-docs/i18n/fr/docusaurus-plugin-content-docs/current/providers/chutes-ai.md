# Utiliser Chutes AI avec Kilo Code

Chutes.ai offre un accès API gratuit à plusieurs modèles de langage volumineux (LLMs), permettant aux développeurs d'intégrer et d'expérimenter avec ces modèles sans engagement financier immédiat. Ils fournissent l'accès à un ensemble.curé de modèles de langage open-source et propriétaires, souvent avec un focus sur des capacités spécifiques ou le support de langues régionales.

**Site Web :** [https://chutes.ai/](https://chutes.ai/)

## Obtenir une Clé API

Pour utiliser Chutes AI avec Kilo Code, obtenez une clé API depuis la [plateforme Chutes AI](https://chutes.ai/). Après vous être inscrit ou connecté, vous devriez trouver une option pour générer ou récupérer votre clé API dans votre tableau de bord de compte ou paramètres.

## Modèles Supportés

Kilo Code tentera de récupérer la liste des modèles disponibles depuis l'API Chutes AI. Les modèles spécifiques disponibles dépendront des offres actuelles de Chutes AI.

Référez-vous toujours à la documentation officielle Chutes AI ou votre tableau de bord pour la liste la plus à jour des modèles supportés.

## Configuration dans Kilo Code

1. **Ouvrir les Paramètres Kilo Code :** Cliquez sur l'icône d'engrenage (<Codicon name="gear" />) dans le panneau Kilo Code.
2. **Sélectionner le Fournisseur :** Choisissez "Chutes AI" dans le menu déroulant "Fournisseur API".
3. **Saisir la Clé API :** Collez votre clé API Chutes AI dans le champ "Clé API Chutes AI".
4. **Sélectionner le Modèle :** Choisissez votre modèle désiré dans le menu déroulant "Modèle".

## Conseils et Notes

- **Accès Gratuit :** Chutes AI fournit un accès API gratuit, en faisant une excellente option pour l'expérimentation et le développement sans coûts immédiats.
- **Variété de Modèles :** La plateforme offre l'accès aux modèles open-source et propriétaires, vous donnant la flexibilité de choisir le bon modèle pour vos besoins.
- **Limites de Débit :** Comme avec tout service gratuit, soyez conscient des limites de débit ou restrictions d'utilisation potentielles qui peuvent s'appliquer à votre clé API.
