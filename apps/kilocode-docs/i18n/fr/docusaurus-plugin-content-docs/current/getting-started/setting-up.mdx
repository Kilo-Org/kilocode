# Configuration de Kilo Code

Lorsque vous vous inscrivez à Kilo Code, vous pouvez commencer immédiatement avec des modèles gratuits, ou recharger votre compte pour la première fois pour obtenir des crédits bonus.

Pour réclamer vos crédits bonus :

1. **Inscription :** Complétez le processus d'inscription
2. **Premier rechargement :** Ajoutez des fonds à votre compte et obtenez 20 $ de crédits bonus
3. **Commencez à coder :** Profitez de vos 20 $ en crédits gratuits

## Processus d'inscription

Kilo Code fournit un processus d'inscription simple qui vous donne accès aux derniers modèles de codage de pointe avec votre connexion Kilo Code.

1. Cliquez sur "Try Kilo Code for Free" dans l'extension
1. Connectez-vous avec votre compte Google sur kilocode.ai
1. kilocode.ai vous demandera d'ouvrir Visual Studio Code
   - Lors de l'utilisation d'un IDE dans un navigateur web, on vous demandera de copier manuellement la clé API à la place
1. Une fois que vous l'autorisez à ouvrir VS Code, vous devez également autoriser VS Code à ouvrir l'URL d'autorisation
