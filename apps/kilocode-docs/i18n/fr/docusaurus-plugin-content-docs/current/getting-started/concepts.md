# Concepts fondamentaux

Découvrez les concepts clés utilisés dans Kilo Code

| Concept                           | Description                                                                                                                                                                                                                                                                                                              | Video Preview                                                                                                                                                                                         |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Fournisseur**                   | Kilo Code fournit son propre fournisseur d'API intégré qui vous donne accès aux derniers modèles de codage de pointe sans avoir besoin de gérer de clés API et qui offre un accès aux modèles de pointe tels que : OpenAI (GPT-5), Anthropic (Claude), Google (Gemini), Meta (Llama) etc.                                | <video width="300" controls><source src="/docs/videos/Provider.mp4" type="video/mp4"/>Démonstration de raisonnement par points de contrôle montrant des modifications de fichiers simultanées</video> |
| **Modèles fondamentaux**          | Modèles IA à grande échelle entraînés sur des vastes ensembles de données qui servent de base aux applications IA. Les modèles comme GPT, Claude, LLaMA fournissent des capacités fondamentales de compréhension et de génération de langage.                                                                            | <video width="300" controls><source src="/docs/videos/Models.mp4" type="video/mp4"/>Modèle fondamental</video>                                                                                        |
| **Ingénierie des invites**        | Art et science de la création d'entrées efficaces pour les modèles IA afin d'obtenir les sorties souhaitées. Utilisez le bouton d'amélioration pour optimiser votre invite                                                                                                                                               | <video width="300" controls><source src="/docs/videos/Prompt.mp4" type="video/mp4"/></video>                                                                                                          |
| **Modes**                         | Dans Kilo Code, vous pouvez choisir le mode **Architecte** pour planifier et concevoir votre logiciel, le mode **Code** pour écrire et refactoriser du code, **Ask** pour comprendre votre base de code, **Debug** pour corriger les problèmes logiciels et **Orchestrateur** pour coordonner les tâches entre les modes | <video width="300" controls><source src="/docs/videos/Modes.mp4" type="video/mp4"/></video>                                                                                                           |
| **Indexation de la base de code** | L'indexation de la base de code permet une recherche sémantique de code dans l'ensemble de votre projet en utilisant des embeddings IA.                                                                                                                                                                                  | <video width="300" controls><source src="/docs/videos/Indexing.mp4"/>Cartographie et navigation de la structure de la base de code</video>                                                            |
| **MCP (Model Context Protocol)**  | Protocole standardisé pour la gestion du contexte et de la communication entre les modèles IA et les systèmes externes. Permet une intégration transparente avec les outils, bases de données et services.                                                                                                               | <video width="300" controls><source src="/docs/videos/MCP.mp4" type="video/mp4"/>Démonstration MCP</video>                                                                                            |
